# 🎮 Joc X și 0 (Tic-Tac-Toe) în C

O implementare clasică a jocului X și 0 (Tic-Tac-Toe) jucat direct în consola Windows. Proiectul este scris în limbajul C și permite la doi jucători să concureze pe același calculator.

## ✨ Funcționalități (Features)
* **Joc în 2 jucători:** Alternare automată a rândului între Jucătorul 1 (X) și Jucătorul 2 (0).
* **Validarea mutărilor:** Sistem care previne suprascrierea unei celule deja ocupate.
* **Sistem de scor:** Ține evidența scorului general pe parcursul mai multor runde.
* **Win Streak:** Calculează seriile de victorii consecutive pentru fiecare jucător.

## 🚀 Cum se rulează

Pentru a juca, ai nevoie de un compilator de C (precum GCC) sau de un mediu de dezvoltare precum Code::Blocks.

**Din terminal / linia de comandă:**
1. Clonează acest repository: `git clone https://github.com/marius20067/x-si-0.git`
2. Compilează fișierul sursă: `gcc main.c -o xsi0`
3. Rulează executabilul: `./xsi0` (pe Linux/Mac) sau `xsi0.exe` (pe Windows).
